Tecnologías



QT Creator

Es un IDE multiplataforma que se ajusta a las necesidades de los desarrolladores Qt. Se centra en proporcionar características que ayudan a los nuevos usuarios de Qt a aprender y comenzar a desarrollar rápidamente, también aumenta la productividad de los desarrolladores con experiencia en la librería.

Django

Es un framework web de Python de alto nivel que fomenta el rápido desarrollo y diseño limpio y pragmático. Construido por los desarrolladores con experiencia, se encarga de gran parte de la molestia de desarrollo Web, para que poder centrarse en la escritura de la aplicación sin necesidad de reinventar la rueda. Es de código abierto y libre.

Ruby on Rails

Es un entorno de desarrollo web de código abierto que está optimizado para la satisfacción de los programadores y para la productividad sostenible. Permite escribir un buen código evitando que se repita y favoreciendo la convención antes que la configuración.

Yii 2.0

Es un framework de PHP rápido, seguro, profesional y de alto rendimiento para el desarrollo de aplicaciones Web.

Node Js

Es un framework y conjunto de librerías basado en JavaScript, usando el motor V8 de Google, permitiendo ejecutar el código de JavaScript de una manera más rápida y realizar la ejecución del mismo del lado del servidor.

AngularJs

Es un framework MVC de JavaScript para el Desarrollo Web Front End que permite crear aplicaciones SPA.

Phonegap

Es un framework para el desarrollo de aplicaciones híbridas, que permite a los programadores desarrollar aplicaciones para dispositivos móviles utilizando JavaScript, HTML5 y CSS3.

Ionic

Es un framework para la creación de aplicaciones móviles con tecnologías web. Es libre y de código abierto, ofrece una librería para móviles optimizada en HTML, CSS y JS para construir aplicaciones altamente interactivas.

Electron

Electron es un framework que permite la creación de aplicaciones de escritorio utilizando JavaScript puro, a través de un runtime con APIs nativas, enfocado en aplicaciones de escritorio, en vez de servidores web, es de código abierto y multiplataforma.

Bootstrap

Es el framework de HTML, CSS Y JS más popular para desarrollo de proyectos "reponsive" en la web.

Aplicaciones Híbridas

Phonegap

Características

  1. Genera aplicaciones móviles con HTML, CSS y JavaScript
  2. Permite comandos por consola
  3. Maneja una API que permite tener acceso a elementos como el acelerómetro, la cámara, los contactos en el dispositivo, la red, el almacenamiento, las notificaciones, etc
  4. Es compatible con framework's de JavaScript
  5. Es un software de código abierto y puede ser utilizado libremente en cualquier aplicación sin necesidad de atribución o licencias de ningún tipo
  6. Con un sólo comando se puede crear, construir, probar y compilar aplicaciones en cualquier plataforma

Ionic

Características

  1. Ionic está construido para ser rápido gracias a la mínima manipulación del DOM, con cero jQuery y con aceleraciones de transiciones por hardware
  2. Ionic utiliza AngularJS con el fin de crear un marco más adecuado para desarrollar aplicaciones ricas y robustas
  3. Una vez desarrollada una aplicación, Ionic permite compilar para disitintos Sistemas Operativos de Móviles
  4. Limpio, sencillo y funcional
  5. Ionic ha sido diseñado para poder trabajar con todos los dispositivos móviles actuales
  6. Con un sólo comando se puede crear, construir, probar y compilar aplicaciones en cualquier plataforma

AngularJS

Características

  1. Patrón de diseño MVC
  2. AngularJS esta continuamente observando los cambios que suceden tanto en la Vista y en el Modelo para sincroniza datos entre estos
  3. Uso de expresiones del tipo: {{ expresión ó valor }} para poder mostrar datos dinámicamente
  4. Uso de directivas para crear elementos HTML y de DOM
  5. Podemos tener un solo elemento HTML (contenedor) dentro de un solo archivo HTML, y dentro de éste poder renderizar cada página que tenga que mostrarse, sin tener que recargar la URL
